前期准备工作

 

在我们正式开始之前,首先请大家准备自己的github 账号和电脑上安装git软件

Git下载地址

正式开始

 

1.安装hugo

 

mac os 安装

brew install hugo

如果你电脑没安装brew工具 ,那就去安装一下就行啦,不懂请百度~

Windows 安装

下载地址 找到windows版本下载即可

2.Hugo是否安装成功

 

安装完成之后,输入hugo version命令查看是否安装成功,如果安装成功会显示版本

hugo version

Hugo Static Site Generator v0.80.0/extended darwin/amd64 BuildDate: unknown

3.使用hugo创建博客

 

hugo 安装成功后，使用hugo new site 命令创建博客

hugo new site myblog

这个命令会创建一个名为myblog 的博客根目录

├── archetypes
│   └── default.md
├── config.toml         # 博客站点的配置文件
├── content             # 博客文章所在目录
├── data                # 数据
├── layouts             # 网站布局
├── static              # 静态文件
└── themes              # 博客主题

4.设置主题

 

博客站点生成后,我们需要给它设置一个美美的主题,主题我们可以去hugo的主题库去下载

找到合适的主题后,我们点击[Download]下载就行啦,一般都会跳转到github 页面,我们用git clone一些就行了

这里我们选用的是m10c这个主题

注意:要在myblog 根目录下载哦

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

下载好主题,我们在 themes 这个文件夹会出现刚才已经下载好的主题

6.编写博客

 

因为这边是需要用Markdown 的格式编写自己的文章,所以前提你需要会用Markdown ,其实这个比较简单,网上很多教程都可以学习.

编写自己的文章后,我们只需要将文件放到 myblog/content/posts 这个文件夹中,hugo 会自动读取这篇文章,将其展示在你的博客中

比如:我们编写了这样一篇文章，文件名为blog1.md

---
title: "我的第一篇文章"   					 #标题
date: 2021-02-03T18:44:17+08:00	 #时间
---
这里是正文

然后将它放到myblog/content/posts 目录下即可

重新我们将hugo 启动.就能看到效果啦

 

7.部署到GitHub

 

在我们把博客建好后,我们就可以将其发布到github上面,别人就可以访问到我们的博客啦

登陆github 点击右上角➕new repository

我们创建的仓库的名字应该是这种格式"账户名.github.io"，

比如我创建的仓库的名字为 ko1a.github.io 切记,千万不能随便写,不然会出现无法访问的问题

创建仓库完成之后.我们就将本地的发布上去

hugo --theme=m10c --baseUrl="https://ko1a.github.io/" --buildDrafts

执行上面这行命令之后,会生成一个public 目录，这个目录中的内容，就是我们博客内容，我们需要将这些内容存放在git 仓库中

 

首先进入public目录  

cd public

 

然后用git 命令push上去

git init
git add .
git commit -m "第一次提交"
git remote add origin https://ko1a.github.io
git push origin master

8.访问公网地址

 

完成之后,我们就将博客系统托管给github了.输入自己的仓库地址直接就可以访问了✌️✌️✌️

https://ko1a.github.io

 如果以后更新博客，则需要再使用

hugo --theme=m10c --baseUrl="https://ko1a.github.io/" --buildDrafts

命令生成新的内容，再将新的内容push 到git 仓库就可以了。
